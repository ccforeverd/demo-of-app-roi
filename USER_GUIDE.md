# 使用说明文档 (USER_GUIDE.md)

## 1. 项目功能介绍

App ROI Dashboard 是一个用于分析应用投放回报率（ROI）趋势的数据看板系统，支持从 CSV 导入数据、按多维条件筛选、查看多周期 ROI 曲线以及预测趋势辅助分析。

核心能力如下：

- 多周期 ROI 可视化：支持 D0 / D1 / D3 / D7 / D14 / D30 / D60 / D90
- 数据导入：支持 CSV 上传并自动写入 MySQL，操作结果以 Toast 通知反馈
- 多维筛选：支持按安装渠道、出价类型、国家地区、APP 过滤
- 图表交互：支持图例开关、悬停详情、预测虚线、回本参考线
- 显示控制：支持原始值/移动平均、线性/对数坐标切换
- ROI 摘要卡片：首页顶部展示 D0/D7/D30 最新 ROI 值及趋势变化
- 深色/浅色主题：支持一键切换，带 0.4s 过渡动画，主题偏好自动保存
- 响应式布局：桌面端侧边栏固定显示，移动端/平板端汉堡菜单触发抽屉式侧边栏

## 2. 数据导入操作指南

### 2.1 启动服务

```bash
# 安装依赖
pnpm install

# 初始化环境变量
cp apps/express/.env.example apps/express/.env
cp apps/nextjs/.env.example apps/nextjs/.env

# 构建共享包
pnpm --filter @demo-of-app-roi/shared build

# 启动 MySQL
docker compose up -d

# 启动前后端
pnpm dev
```

常用地址：

- 前端页面：`http://localhost:3000`
- 后端 API：`http://localhost:3001`
- Swagger 文档：`http://localhost:3001/api-docs`

### 2.2 CSV 导入方式

方式一：命令行导入

```bash
curl -X POST http://localhost:3001/api/roi/import \
  -F "file=@example/app_roi_data.csv"
```

方式二：Swagger UI 导入

1. 打开 `http://localhost:3001/api-docs`
2. 找到 `POST /api/roi/import`
3. 点击 `Try it out`
4. 选择 CSV 文件并执行

### 2.3 CSV 格式要求

| 列名 | 说明 | 示例 |
|---|---|---|
| 日期 | `YYYY-MM-DD(星期)` | `2025-04-13(日)` |
| app | 应用名称 | `App-1` |
| 出价类型 | 出价模式 | `CPI` |
| 国家地区 | 国家名称 | `美国` |
| 应用安装.总次数 | 安装数量 | `4849` |
| 当日ROI ~ 90日ROI | 百分比值 | `6.79%` |

### 2.4 清空数据（仅开发环境）

需确保 `apps/express/.env` 中 `NODE_ENV=development`：

```bash
curl -X DELETE http://localhost:3001/api/roi/clear
```

返回示例：

```json
{
  "success": true,
  "data": {
    "deleted_rows": 910,
    "remaining_rows": 0
  },
  "error": null
}
```

## 3. 图表操作指南

### 3.1 ROI 摘要卡片

图表上方展示 D0、D7、D30 三个关键 ROI 维度的最新值摘要卡片：

- 显示最新一条有效数据的 ROI 百分比值
- 趋势箭头：与前一条数据比较，上升显示绿色向上箭头，下降显示红色向下箭头
- 鼠标悬停卡片有轻微上浮效果

### 3.2 图表交互

图表区域默认展示多周期 ROI 折线，支持以下交互：

- 鼠标悬停：查看当前日期各 ROI 周期具体数值
- 图例点击：显示/隐藏对应曲线
- 预测识别：实线表示实际值，虚线表示线性外推预测值
- 回本判断：红色水平线为 100% ROI 回本参考线
- 数据缩放：支持触摸/滚轮缩放（DataZoom）

使用建议：

- 先查看摘要卡片了解最新 ROI 概况
- 再查看整体趋势，通过图例逐条排查异常周期
- 对波动较大的数据优先使用”移动平均值”观察趋势
- ROI 跨度较大时切换”对数刻度”便于观察低值区间

## 4. 筛选功能使用方法

页面左侧侧边栏”筛选条件”区域提供四个核心筛选器：

1. 用户安装渠道
2. 出价类型
3. 国家地区
4. APP

使用方式：

- 选择任意筛选项后，图表会自动触发数据刷新
- 选择”全部”可取消该维度筛选
- 可多条件叠加筛选（例如：渠道 + 国家 + APP）

推荐筛选路径：

- 先选 APP（聚焦单产品）
- 再选国家/渠道（定位投放场景）
- 最后切换出价类型（比较投放策略差异）

> 移动端/平板端：点击左上角汉堡菜单图标打开侧边栏

## 5. 控制器使用说明

控制器位于侧边栏”显示设置”区域，分为两个区域：

### 5.1 数据显示模式

- 显示移动平均值：默认 7 日平滑，适合看趋势
- 显示原始数据：保留每日波动，适合看细节

### 5.2 Y 轴刻度

- 线性刻度：适合数据范围接近时对比
- 对数刻度：适合数值跨度大时压缩高值、放大低值

## 6. 主题切换

点击页面右上角的太阳/月亮图标可切换深色和浅色主题：

- 深色主题（默认）：深蓝色背景，适合夜间使用
- 浅色主题：浅灰色背景，适合日间使用
- 主题切换带有 0.4 秒平滑过渡动画
- 主题偏好自动保存到浏览器，下次访问自动恢复

## 7. 数据管理

侧边栏”数据管理”区域提供 CSV 上传功能：

- 点击”上传 CSV 文件”按钮选择文件
- 上传中按钮显示旋转加载图标
- 上传成功/失败会在右上角弹出 Toast 通知（3 秒后自动消失）
- 上传成功后图表和摘要卡片自动刷新
